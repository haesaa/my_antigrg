# C3: 시네마토그래피 및 비주얼 연출 (SoT)

- **카메라 구도**: ELS(광활) -> LS -> MS -> CU -> ECU(부분 강조). 감정선에 따라 점진적 줌
- **조명 (Lighting)**: 
  - 키 라이트/백 라이트로 인물 부각
  - 분위기: 고독(Cool Blue/Low-key), 로맨스(Warm Amber/Golden Hour)
- **움직임**: 트래킹, 팬, 틸트. 속도(BPM)에 맞춘 컷 전환
- **프롬프트 네거티브 지침**: "text, watermark, deformed, blurry" 등 기본 네거티브 필수 포함
